
# MyQart Ecommerce



## Tech Stack

**Client:** Angular, NGRX

**Server:** Node, Express

**Database**: MongoDB


## Run Locally

Clone the project

```bash
  git clone https://github.com/crstnmac/MyQart
```

Go to the project directory

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
